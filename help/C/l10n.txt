SCREENSHOT INSTRUCTIONS
------------------------

gcalctool_basic_window.png

1. Start Calculator (Applications -> Accessories -> Calculator).
   To check that you have started gcalctool, and not gcalc, choose Help > About.
2. Choose View > Basic Mode.
3. Ensure that the window has focus. If it does not, click on the window to 
   give it focus.
4. Take a screenshot of the window exactly as it is when you start the 
   application.
5. Save the file in PNG format.


gcalctool_financial_window

1. Start Calculator (Applications -> Accessories -> Calculator).
   To check that you have started gcalctool, and not gcalc, choose Help > About.
2. Choose View > Financial Mode.
3. Ensure that the window has focus. If it does not, click on the window to 
   give it focus.
4. Take a screenshot of the window exactly as it is when you start the 
   application.
5. Crop the section containing the Financial Mode buttons.
6. Save the file in PNG format.


gcalctool_scientific_window.png

1. Start Calculator (Applications -> Accessories -> Calculator).
   To check that you have started gcalctool, and not gcalc, choose Help > About.
2. Choose View > Scientific Mode.
3. Ensure that the window has focus. If it does not, click on the window to 
   give it focus.
4. Select the Hex option.
5. Take a screenshot of the window.
6. Crop the section containing the Scientific Mode buttons and the Scientific 
   Mode options.
7. Save the file in PNG format.

-------------------------
GUI* TAGS AND SUPERSCRIPT
-------------------------

Several Calculator buttons have a superscript label.
Currently, the <superscript> tag is not allowed within a <guibutton>, 
<guilabel>, or <guimenu> tag.
Therefore, any references to such Calculator buttons are currently not embedded 
in <gui*> tags.
